# Rare Plant Atlas

## Content notes

### Cinematic panels are visual-first
The `panels` content should focus on how the plant looks and feels in person. Prioritize visual qualities like leaf texture, variegation pattern, color contrast, shape, and flower presence. These sections are intentionally image-heavy, so copy should support what viewers are seeing (for example: dramatic foliage, beautiful leaves, striking blooms) rather than broad care instructions.
